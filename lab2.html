Lab 2: IMU

Objective

The purpose of this lab is to integrate an IMU into the robot, transition to battery-powered operation, and record a stunt with the RC robot.

Parts Required

  • 1 x SparkFun RedBoard Artemis Nano
  • 1 x USB cable
  • 1 x 9DOF IMU sensor
  • 1 x Force1 RC car
  • 1 x Li-Ion 3.7V 850mAh battery

Instructions

Setup the IMU

  • Install the “SparkFun 9DOF IMU Breakout_ICM 20948_Arduino Library” from the Arduino Library Manager.
  • Connect the IMU to the Artemis board using the QWIIC connectors.
  • Run the example program and analyze sensor output.
